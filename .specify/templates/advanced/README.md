# Advanced Template Set

This folder provides an optional wave-based planning workflow adapted from
`../ultimate-project` for use with Rustchat.

Use these templates when a feature needs stricter execution control:
- pre-implementation gates
- wave-based delivery
- explicit constitution and compatibility checks

Templates:
- `plan-template.md`
- `spec-template.md`
- `tasks-template.md`

These are additive and do not replace default templates in
`.specify/templates/`.
